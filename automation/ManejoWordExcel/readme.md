# Ejercicios de automatizacion con Pandas y manejo de archivos de Word y excel
Este lo tome de un tutorial de youtube de [Código Espinoza - Automatiza tu Vida](https://www.youtube.com/watch?v=vjACmOQe6yQ)

Descripcion de los archivos
* app.py Genera un documento word con un parrafo sencillo
* app2.py Genera un documento word con diferentes estilos en diferentes parrafos
* app3.py de un archivo excel usando pandas lo agrega a un documento de word
* word_automation Libreria pequeña que engloba los ejercicios anteriore para ser reutilizable
* main Ejecuta la libreria anterior generando un word con todo lo anterior
